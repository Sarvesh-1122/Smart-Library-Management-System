# 📚 Smart Library Management System

A Java Full Stack Library Management System developed using JSP, Servlets, JDBC, MySQL, HTML, CSS, and JavaScript.

---

## 🚀 Features

- 🔐 Admin & Student Login
- 👨‍🎓 Student Registration
- 📚 Book Management
- 📂 Category Management
- 📖 Book Issue
- ✅ Book Return
- 🔍 Search Books
- 📊 Admin Dashboard
- 📈 Student Dashboard
- 💾 MySQL Database Integration

---

## 🛠️ Tech Stack

### Backend
- Java
- JDBC
- JSP
- Servlets

### Frontend
- HTML
- CSS
- JavaScript

### Database
- MySQL

### Server
- Apache Tomcat 10

### IDE
- Eclipse IDE
